Breath: Empowering Asthma Management through Technology

As the visionary group leader of a remarkable university project, I am thrilled to present "Breath," an innovative asthma management application that merges groundbreaking research with cutting-edge technology. Breath is a powerful tool that puts individuals in control of their asthma, offering a higher quality of life.

See the Code

Key Features:

  • Health Education: A comprehensive knowledge hub with in-depth research-backed content, empowering users with the knowledge to make informed decisions about their asthma care.
  • Patient Diary: An intuitive diary for effortless tracking of asthma-related activities, fostering better communication with healthcare providers for personalized care.
  • Self-Monitoring Appointment: Smart reminders and notifications to ensure users never miss crucial healthcare appointments or medication schedules.
  • Inhaler Usage Monitoring: A user-friendly inhaler usage tracker, providing proactive alerts for timely refills and optimal asthma control.
  • Medication Reminder: Timely notifications promoting adherence to prescribed schedules, crucial for preventing asthma exacerbations.
  • Insights into Asthma Attacks: Valuable data on the frequency and patterns of asthma attacks, empowering users and healthcare providers to optimize treatment plans for better well-being.

Breath boasts an exceptional user interface, skillfully crafted by our talented team, guaranteeing a seamless and engaging experience. From concept to meticulous implementation, our project showcases the synergy of research, innovation, and dedication.

Tools and Technologies:

  • Mobile App Development: React Native
  • Backend Development: Node.js
  • API Integration: Axios API (for fetching data from server), OpenWeatherMap API (for weather data)
  • Cloud Services: AWS S3 (Simple Storage Service) for media file storage, AWS RDS (Relational Database Service) for managing data, Glitch server (for hosting server)
  • Collaboration & Development Tools: Visual Studio Code (as the primary code editor for the project), Adobe XD (for UI/UX design)

This significant project involved designing and developing an innovative application, serving as the final requirement for my bachelor's degree. As the team leader, I guided a group of six members throughout the project, using Adobe XD for design, React Native for frontend, and Node.js for the backend. The project received widespread acclaim, and our team was awarded the first prize for our outstanding achievements.

Password Security Enhancement Using Bloom Filters

In this comprehensive security project, I leveraged advanced techniques like Bloom filters, hash functions, and similarity metrics to tackle critical password security challenges. By applying robust methodologies, I successfully addressed common weaknesses in password selection and storage.

See the Code

Key Features:

  • Utilized 15 Hashing Functions: To ensure strong password protection.
  • Employed Salting Techniques: To safeguard password files from unauthorized access.
  • Implemented Bloom Filters: To efficiently check for common and weak passwords during user registration.
  • Applied Cryptographic Hashing Functions: MD5 and SHA256 to enhance the security of stored passwords.
  • Utilized Bigrams and Jaccard Coefficients: To measure password similarities and identify potential threats.

Features:

Security Measures:

  • Salting
  • Hashing Functions
  • Cryptographic Hashing

Testing and Implementation:

  • Testing on Datasets

Development Tools:

  • Java
  • Replit.io

Achievement:

This project was done for the System Security course's final assignment. The objective was to create a system that prevents users from using frequently used passwords. I used Java to find a solution, generating random strings of 20 characters as salt and creating new passwords that combine salt and user passwords to store them securely. The project met all the requirements and received a score of 92%.

Results: Scored 92%
